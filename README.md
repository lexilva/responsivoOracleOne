

# 📱 **Design Responsivo**  

Este projeto foi desenvolvido para demonstrar as melhores práticas em **design responsivo**, garantindo que o layout se adapte perfeitamente a diferentes dispositivos e tamanhos de tela.  

## 🚀 **Tecnologias Utilizadas**  

- **HTML5** → Estrutura semântica  
- **CSS3** → Estilização e responsividade  
- **JavaScript** (opcional) → Interatividade dinâmica  
- **Media Queries** → Ajustes específicos para diferentes telas  
- **Flexbox & CSS Grid** → Construção de layouts flexíveis e adaptáveis  

## 🎯 **Principais Funcionalidades**  

✔️ **Layout fluido e responsivo**  
✔️ Uso de **rem, em, vw, vh** para medidas escaláveis  
✔️ **Imagens adaptáveis** (`max-width: 100%`) para melhor desempenho  
✔️ **Media Queries otimizadas** para diferentes resoluções  
✔️ **Testado em múltiplos navegadores e dispositivos**
